# javascript-utils

A collection of JavaScript utility functions.

## dom-parser.js

This utility demonstrates how to parse an HTML string and extract `href` attributes from `<a>` tags using `jsdom`.

### Dependencies

- `jsdom`

### Usage

The `src/dom-parser.js` file currently contains an example of parsing a static HTML string and logging the extracted links to the console.

To run the example:

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Run the script:
   ```bash
   node src/dom-parser.js
   ```

## Installation

To set up the project, clone the repository and install dependencies using pnpm:

```bash
git clone <repository-url>
cd javascript-utils
pnpm install
```
