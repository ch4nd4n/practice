## Project Overview

This project, `javascript-utils`, is a collection of various JavaScript utility functions. The initial utility, `dom-parser.js`, focuses on HTML parsing and data extraction.

## Key Files

- `src/dom-parser.js`: Contains functionality for parsing HTML content using `jsdom` and extracting information, specifically `href` attributes from anchor tags.

## Dependencies

- `jsdom`: A JavaScript implementation of the W3C DOM and HTML standards, used for parsing and interacting with HTML.

## Preferred Package Manager

The preferred package manager for this project is `pnpm`.
