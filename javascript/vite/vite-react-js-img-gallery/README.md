# Vite React Image Gallery

A simple Image Gallery application built with Vite and React.

## Features
- **Landing Page**: Entry point to the application.
- **Image Gallery**: Search and filter through a collection of images.
- **Documentation**: See `docs/` for detailed requirements.

## Setup

1. **Install Dependencies**
   ```bash
   pnpm install
   ```

2. **Run Development Server**
   ```bash
   pnpm run dev
   ```

## Structure
- `src/components`: Contains React components.
- `docs/`: Project documentation and requirements.
