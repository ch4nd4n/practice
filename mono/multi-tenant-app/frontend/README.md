
# **Frontend: React & Vite**

## **Setup**

1. `npm install`.  
2. Configure `.env` with `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.

## **Development Flow**

* **Run**: `npm run dev`.  
* **E2E Tests**: `npx playwright test`.

## **Rules**

* Use generated API clients from the backend spec.  
* All routes protected by Supabase Auth guards.
